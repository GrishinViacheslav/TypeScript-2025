Система управління сповіщеннями Клас: NotificationService Складний тип: Визначення типу NotificationPayload. EmailPayload = { recipient: string, subject: string } SMSPayload = { phone: string, text: string } Об'єднання та Перетин: NotificationPayload = ({ type: 'email' } & EmailPayload) | ({ type: 'sms' } & SMSPayload) (Дискримінаційне об'єднання для безпечної роботи з різними типами сповіщень) Метод: sendNotification(payload: NotificationPayload)
